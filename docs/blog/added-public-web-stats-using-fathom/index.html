Added public web stats using Fathom

2019-02-07

Added (GDPR compliant) stats generation today for the website, using Fathom. It’s an Open Source project, written in Go, that aims to provide aggregated stats simply, without any user tracking.

The setup was fairly easy, though the ARM64 download for the current release (1.2.1) doesn’t seem to work with SQLite due to an accidental setting during their build. Hopefully fixed for the next release.

PostgreSQL is being the database, and Nginx providing HTTPS termination.

The stats server is public, so added a link to it from our menu bar just because.
