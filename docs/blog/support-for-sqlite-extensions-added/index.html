Support for SQLite extensions added

2023-05-04

On DBHub.io, we’ve just rolled out support for these extensions, for all database types:

  • FTS3 & FTS3_PARENTHESIS
  • FTS5
  • GEOPOLY
  • JSON1
  • MATH_FUNCTIONS
  • RTREE
  • SOUNDEX

This means your databases are now able to use all of the functions provided by these extensions, which should be very useful for a lot of people.
