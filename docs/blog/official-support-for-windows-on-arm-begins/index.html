Official Support for Windows on ARM Begins

2026-04-06

Hello Community, We have some exciting news to share today.

Our project has officially begun supporting Windows on ARM. Those interested in this ARM64 native build can download it following: https://github.com/sqlitebrowser/sqlitebrowser/releases/tag/v3.13.1

- DB.Browser.for.SQLite-v3.13.1-arm64.msi
- DB.Browser.for.SQLite-v3.13.1-arm64.zip

Please note that since we are using Qt6-rather than Qt5-as the underlying framework to support ARM64 native builds, there may be potential issues that have not yet been identified. We therefore ask for your continued feedback. :)

Thank you.
